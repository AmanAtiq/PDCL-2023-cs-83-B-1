# Lab 4: Advanced Collectives & Performance Analysis
Deep dive into reduction operations and performance benchmarking.

## Key Exercises
- **MPI_Allreduce & MPI_Scan:** Computing global sums and prefix sums.
- **Manual vs. Collective:** Comparing a manual loop-based broadcast against the optimized `MPI_Bcast`.
- **Timing:** Using `MPI_Wtime` to measure communication overhead.
